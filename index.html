BULK FRUIT/VEGETABLE ORDERING PLATFORM

This project is a full-stack web application built for bulk ordering of vegetables and fruits. Buyers can browse products, place bulk orders, and track the status of their orders. Admins can manage orders and update their statuses, and also add, edit, or remove products from the catalog. The application includes features like email notifications for order status updates. Built using React, Redux, Node.js, Express, MongoDB, and JWT authentication, it provides a complete and efficient platform for buyers and admins.
